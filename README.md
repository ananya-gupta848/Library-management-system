Project Scope-It manages book cataloging, user registration, issue/return processes, fines, and reports to streamline library operations efficiently.

DOCUMENTATION:-

## ⚙️ Features
- **[Add Book](ca://s?q=Explain_Add_Book_feature_in_Library_Management_System)** – Allows adding new books with ID, title, and author.  
- **[Add User](ca://s?q=Explain_Add_User_feature_in_Library_Management_System)** – Registers new users with unique IDs and names.  
- **[Issue Book](ca://s?q=Explain_Issue_Book_feature_in_Library_Management_System)** – Issues available books to registered users.  
- **[Return Book](ca://s?q=Explain_Return_Book_feature_in_Library_Management_System)** – Handles book returns and applies fines if returned late.  
- **[Reserve Book](ca://s?q=Explain_Reserve_Book_feature_in_Library_Management_System)** – Allows users to reserve books that are not currently issued.  
- **[Show Report](ca://s?q=Explain_Show_Report_feature_in_Library_Management_System)** – Displays the status of all books and users, including fines.
